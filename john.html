My Project

1. Multi Purpose E-card System using LPC2129.

The Multi-Purpose eCard System integrates EEPROM (AT24C256) for secure data storage, RFID reader for authentication, and RTC for validity checking. Stored data includes Aadhar, PAN, license, and voter ID, accessible only with the correct password. RFID authentication ensures card legitimacy, while real-time validity checking denies access to expired cards. Hardware components communicate seamlessly, employing SPI for EEPROM and UART for RFID. With encryption and strict access controls, the system guarantees confidentiality and integrity of stored data. This versatile solution caters to identity management needs, offering security, efficiency, and ease of use in accessing personal identification information.

2.Arithmetic Calculator Project using LPC2129.

Designed and implemented an Arithmetic Calculator utilizing the LPC2129 microcontroller with ARM7TDMI-S core. The project features a user-friendly interface with a 16x2 LCD display and keypad for basic arithmetic operations, including addition, subtraction, multiplication, and division. Leveraged efficient memory management for operand storage and incorporated real-time capabilities for instant calculation and display of results. The project showcases error-handling mechanisms to ensure accurate calculations and operates with low power consumption, making it suitable for educational purposes and as a practical learning tool for embedded systems development.

3.Rain Protected Grain Project.

Conceived and executed an innovative solution for safeguarding grain heaps during rain by designing and implementing an automated covering system. Leveraged Arduino Nano and rain-detecting sensors to detect precipitation and deploy protective covering sheets automatically. This project demonstrates my proficiency in embedded systems, sensor integration, and automation, showcasing a practical application of technology for agricultural preservation.
